# 会员制医疗预约服务管理信息系统

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=11002

## 技术介绍
Java语言 + ssm框架/springboot框架（后端）+ Vue前端框架（Web）+ Mysql(数据库)

## 开发工具
IDEA或者MyEclipse + Navicat(mysql数据库可视化工具) + Tomcat

## 项目视频演示
- [演示视频1](https://graduation-images.oss-cn-beijing.aliyuncs.com/videos/94%E5%A5%97-2-springboot%E5%BD%95%E5%83%8F/11002_springboot%E4%BC%9A%E5%91%98%E5%88%B6%E5%8C%BB%E7%96%97%E9%A2%84%E7%BA%A6%E6%9C%8D%E5%8A%A1%E7%AE%A1%E7%90%86%E4%BF%A1%E6%81%AF%E7%B3%BB%E7%BB%9F%E6%BC%94%E7%A4%BA%E5%BD%95%E5%83%8F2022.mp4)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
